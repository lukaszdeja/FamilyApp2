# FamilyApp
Projekt powstał na pierwszym hackatonie programistycznym w jakim nasz zespół brał udział w technikum, w 2021 roku.
Aplikacja PWA wspierająca rodziny, współpracę i komunikację.

Zespół projektowy i podział zadań:
1. Łukasz Deja - backend oraz integracja z frontendem i bazą danych
2. Sebastian Madzia - integracja backendu z wykorzystanym API.
3. Szymon Kosecki - frontend
4. Artur Szymański - frontend
